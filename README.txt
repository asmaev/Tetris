Projet de programmation impérative
But : Faire un Tetris
Noufoussi Asmae
Dufaud Noah


Ouverture du TETRIS :
  - Ouvrir un terminal à cet emplaçement (c'est à dire dans le dossier 'Tetris').
  - Entrer la commande 'make'.
  - Executer ./Tetris, ou simplement Tetris
