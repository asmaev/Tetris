Imperative Programming Project
Aim : Create a Tetris
By: Noufoussi Asmae & Dufaud Noah


How to open TETRIS :
  - Open a terminal in the Tetris folder.
  - Enter the command 'make'.
  - Run ./Tetris, ou simply Tetris
